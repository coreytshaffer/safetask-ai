## Project Status and Safety Framing
* **Status**: Prototype. SafeTask-AI is a research-grade structural prototype to prove that local hazard processing can exist independently of surveillance capabilities.
* **Safety Framing**: This repository establishes strict boundaries for ingestion, schema validation, human review, and export redaction before any actual visual models or live cameras are connected.
* **What the repo should NOT be described as**:
  * "Open-source CCTV system"
  * "AI security camera"
  * "Law enforcement automation tool"
  * "Surveillance or tracking system"
  * "Production security system"
