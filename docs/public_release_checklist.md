# Public Release Checklist

Before changing visibility to Public or tagging a release, verify the following:

- [x] README clearly marks prototype status
- [x] No production security claims
- [x] No life-safety claims
- [x] No real footage
- [x] No real incidents
- [x] No private camera/vendor references
- [x] No household/security details
- [x] No private IPs except loopback/dev addresses
- [x] No secrets/credentials
- [x] No real faces, plates, addresses, or people
- [x] Synthetic fixtures only
- [x] No ALPR
- [x] No face recognition
- [x] No person re-identification
- [x] No tracking
- [x] No automated law-enforcement workflows
- [x] No automated EMS/911 workflows
- [x] Tests pass
- [x] `compileall` passes
- [x] `git diff --check` passes
- [x] License selected (MIT)
- [x] GitHub description prepared
- [x] Topics prepared
- [x] Release tag prepared
