# Safety Boundaries

The PitLens Table Games Review Workbench is framed strictly as authorized review support for surveillance, QA, training, and report preparation.

## Core Constraints

1. **No Accusations:** The tool does not accuse players or staff. It does not generate conclusions of "cheating." All findings must preserve human authority.
2. **Reviewer-Confirmed Facts:** Reviewer-confirmed facts are strictly distinguished from system estimates. The software assists humans, not replaces them.
3. **Sanitized Anomaly Examples:** Anomaly labels are sanitized. E.g., we use "wager timing anomaly" or "procedure exception," avoiding language that describes specific cheating methods in a way that teaches execution.
4. **Controlled Footage Use:** Any controlled scenario footage must not become public instructional content.
5. **No Cheating Tutorials:** The system must not generate operationally useful cheating tutorials.
