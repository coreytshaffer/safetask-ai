# SafeTask AI Integration Boundary

This document outlines how PitLens could integrate into SafeTask AI in the future.

## Responsibilities

- **PitLens:** Owns game logic, presets, active bet shot status, wager analysis, strategy guidance, and the table-games timeline.
- **SafeTask AI:** Owns case files, evidence retention, policy references, role access, incident reports, and broader surveillance workflows.

## Integration Path

- Integration should occur through JSON export/import or API interfaces later.
- The current MVP operates independently and does not require SafeTask AI to run.
