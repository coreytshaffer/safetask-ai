# PitLens: Table Games Review Workbench

## Project Purpose

PitLens is a local-first Table Games Review Workbench designed to run as either a standalone product or as a future module inside **SafeTask AI** called "Table Games Monitoring Mode". The MVP focuses on authorized casino surveillance, table-games review, training, QA, and report preparation.

## Local-First Boundary

To maintain compliance and data security, this tool relies on local-first storage and processing. It does not send footage, screenshots, or sensitive casino data to cloud services by default.

## MVP Scope

The first product wedge is **Blackjack Review Mode**, specifically bet-spread tracking, active bet shot confirmation, and basic-strategy deviation review. It is intended to assist human reviewers and does not present automated conclusions as proof of cheating.
