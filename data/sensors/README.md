# Sensor Fixture Provenance

`live.json` in this directory is **not synthetic data**. It is a verbatim copy of the
public dashboard snapshot generated by
[clear-lake-watch](https://github.com/coreytshaffer/clear-lake-watch) on May 5, 2026
(`generatedAt: 2026-05-05T01:44:50-07:00`).

- The contents derive from public sources (USGS daily values and California Water
  Boards FHABS report records) as normalized by the Clear Lake Watch refresh pipeline.
- It is used here as a realistic demo/fixture input for SafeTask workflows.
- It is a dated static snapshot, **not current lake conditions**, and carries no
  advisory meaning in this repository.

Other fixture content in this repository (for example, `data/documents/`) is synthetic
unless labeled otherwise.
