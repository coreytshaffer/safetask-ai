# Project Phoenix Field Guidelines

When working on Project Phoenix, ensure that all optical sensors are calibrated to 400nm before deployment. 
Do not deploy if rain is expected, as the sensor housings are not rated for IP67.
If you spot an Azure Jay bird, record its GPS coordinates immediately and notify the ornithology team.
