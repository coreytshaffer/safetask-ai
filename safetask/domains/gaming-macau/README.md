# SafeTask Gaming Macau Pack

This folder is reserved for a future Macau gaming regulatory source pack.

Macau should be treated as a multilingual jurisdiction pack, not as a translation of the Nevada or tribal gaming packs. Entries should preserve original source language text and separately track literal translation, operational interpretation, reviewer status, and user-facing language.

Source acquisition is official public sources only. Use Macao SAR Government Portal, DICJ, or MSAR Official Gazette / legislation pages as the starting source roots.

Pack entries must follow the `official_source_plus_reviewed_operational_interpretation` model:

1. Preserve official source text.
2. Add human-reviewed literal translation.
3. Add reviewer-approved operational interpretation for surveillance users.

Draft entries can be stored with `confidence: needs_review`, but they are not operator-facing guidance until the pack passes approved validation.
